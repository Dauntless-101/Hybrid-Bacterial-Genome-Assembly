# Contributing

Thank you for considering contributing to **Hybrid-Bacterial-Genome-Assembly**!

## How to contribute

1. **Report bugs** or suggest features by opening an issue.
2. **Improve documentation** — even small fixes help.
3. **Submit pull requests** for code changes.

## Development setup

Clone the repository and create the root Conda environment:

```bash
git clone https://github.com/Dauntless-101/Hybrid-Bacterial-Genome-Assembly.git
cd Hybrid-Bacterial-Genome-Assembly
conda env create -f environment.yml
conda activate hybrid-assembly
